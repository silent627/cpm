# 对话框拖拽和缩放指令

## 功能说明

`v-dialog-drag` 指令为 Element Plus 的 `el-dialog` 组件添加了拖拽和缩放功能。

## 功能特性

1. **拖拽功能**
   - 点击对话框标题栏可以拖拽对话框
   - 自动限制在窗口范围内
   - 鼠标悬停在标题栏时显示移动光标

2. **缩放功能**
   - 对话框右下角显示缩放手柄
   - 拖拽缩放手柄可以调整对话框大小
   - 最小尺寸：400x300
   - 最大尺寸：窗口大小减去边距

## 使用方法

在 `el-dialog` 组件上添加 `v-dialog-drag` 指令：

```vue
<el-dialog v-model="dialogVisible" title="对话框标题" v-dialog-drag>
  <!-- 对话框内容 -->
</el-dialog>
```

- 使用 Vue 3 自定义指令
- 监听鼠标事件实现拖拽
- 动态创建缩放手柄元素
- 自动清理事件监听器
